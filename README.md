# NTTData-Webapp

### Remove all container from docker

```
sudo docker rm -f (sudo docker ps -aq)
```

### Run project

```
sudo docker compose up
```

### Add permission create folder and files (Linux)

```
sudo chmod -R 777 webapp2
```
